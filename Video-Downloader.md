# Youtube Video Downloader

---
This is a **Youtube Video downloader** based on **[yt-dlp](https://github.com/yt-dlp/yt-dlp)**.
It has custom UI made with **[customtkinter](https://customtkinter.tomschimansky.com/)**.

On Paper it just extracts information from Youtube, downloads the Video and merges them together with **[ffmpeg](https://www.ffmpeg.org/)**, a popular video/audio tool. 

If ffmpeg is not installed or not availible it gets **downloaded automaticly**, which can take up a few minutes on Start-up. This is because the module is necessary for the program and will not function without it.
